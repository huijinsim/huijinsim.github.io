# Forest Soul Portfolio

An immersive, interactive 3D portfolio experience built with Next.js and Three.js.

## Key Features

- **Interactive Forest Intro**: Users can plant procedural 3D flowers and trees that grow dynamically on the landing page.
- **Atmospheric Weather System**: Double-click the sky to trigger a gentle, atmospheric rain effect with real-time lighting transitions.
- **3D Manifesto Visualization**: Interactive components that visualize design philosophy through growing botanical structures.
- **Refined Typography**: Unified modern sans-serif aesthetic for a professional and premium feel.
- **Responsive Design**: Fully optimized for various screen sizes with smooth transitions and micro-animations.

## Tech Stack

- **Framework**: Next.js (App Router)
- **3D Graphics**: Three.js
- **Animations**: Motion (Framer Motion)
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view the result.
